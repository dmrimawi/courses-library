	. if grade >= 60 then
		o Increment pass
	. else
		o Increment fail
	. end if
	. calculate sum = sum + grade
	. Ask user to enter the grade
	. Input grade
- End while
- print fail
- print pass
- compute failure = fail / (fail + pass)
- print failure

- end
